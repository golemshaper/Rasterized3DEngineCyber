# Rasterized3DEngineCyber

A game written in SDL and C++ built with `conan` and `cmake`.

`cmake` is a built tool used to compile and link C/C++ Programs.

`conan` is a Package Mangager for C/C++ which can download any third party libraries from the internet automatically. It then
generates any required files for the build, such as the `Makefiles`, that `cmake` can then execute.

## Releasing
 
A production build can be done with `Python` and `cmake`

### Prerequisites

- [cmake](https://cmake.org/download/)
- [Python](https://www.python.org/downloads/)

### First Time Setup

> **NOTE**: This step only needs to be done one time when you first clone the repo.

After installing `cmake` and `Pythonn`, you must run a script which sets up Python and the package manager `conan`:

Windows:

```bat
.\scripts\setup.bat
```

Unix:

```bash
./scripts/setup.sh
```


### Running a Release

To run a release, simply run the `release.py` script:

Windows:

```sh
.\release.py
```

Unix:

```sh
./release.py
```

After the script completes, run the game with

Windows:

```bat
.\build\Rasterized3DEngineCyber
```

Unix:

```sh
./build/Rasterized3DEngineCyber
```
